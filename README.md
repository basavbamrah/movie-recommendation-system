# Movie Recommendation System
This is a movie recommendation system that uses the MovieLens dataset. The dataset contains 100,000 ratings from 1000 users on 1700 movies. The dataset can be found [here](https://grouplens.org/datasets/movielens/100k/).



***Note***: This project is still in development. Recommendation system is ready and working. The web app is still in development.
for recommendation system look at the **trainModel folder.**
